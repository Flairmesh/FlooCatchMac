# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-27

### Added

- initial public release of the FlooCatch macOS receiver app
- automatic discovery of supported FlooGoo FMA120 USB CDC devices
- broadcast scanning, selection, sync, and stop controls
- audio playback routing to the selected macOS output device
- encrypted broadcast PIN entry flow
- localized user interface for multiple languages
- Developer ID signing and notarized release packaging

### Notes

- minimum supported macOS version is 13.0
- distributed outside the Mac App Store as a notarized app
