# FlooCatchMac v1.0.0

First public release of FlooCatchMac.

## Highlights

- Discover and connect to supported FlooGoo FMA120 receiver hardware on macOS
- Scan for Auracast broadcasts and tune in with a simplified end-user UI
- Enter PIN codes for encrypted broadcasts when prompted
- Route received audio to the selected macOS output device
- Use the expert panel for receiver state, manual commands, and session logging

## Requirements

- macOS 13.0 or later
- FlooGoo FMA120 receiver hardware

## Distribution

- signed with Developer ID
- notarized by Apple for direct website download
